앱 주요 기능
1. 방문 차량 예약
- 방문 예정일, 차량 번호, 방문 목적 등을 입력하여 예약.
2. 예약 내역 조회
- 예약한 차량의 내역을 날짜나 차량번호로 검색하여 확인.
3. 방문 차량 기록 확인
- 예약 후 방문한 차량의 기록을 확인.
4. Q&A
- 방문 차량 예약 방법, 예약 내역 확인 방법, 방문 기록 확인 방법, 탈퇴 방법 등.
